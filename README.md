## 🌌 The Vision
**COS** isn't just a suite of tools; it's a **cognitive extension** for the modern digital worker. By seamlessly integrating into your operating system, COS monitors your context, understands your intentions, and proactively restores your focus before you even realize you've lost it.

---

## ✨ Features

### 📡 **Neural Context Engine**
*Stop searching, start recalling.* COS captures browser states, active window metadata, and even real-time OCR from your screen to build a high-fidelity semantic memory of your work.
- 🖼️ **Real-time OCR**: Scans your screen to understand visual context.
- 🕒 **Context Timeline**: A perfect memory of everything you've worked on.
- 🔄 **One-Click Resume**: Instantly restore workspaces, URLs, and VS Code environments.

### 🕸️ **Knowledge Graph Visualization**
*See your mind in action.* Your tasks aren't isolated; they're interconnected. COS clusters your activities into high-level "Projects" and visualizes them in a stunning, interactive 3D graph.
- 🔭 **Cluster Analysis**: Automatic grouping of related contexts.
- 🔗 **Relationship Mapping**: See how your research connects to your code.

### 🎙️ **Voice Command & Proactive Autonomy**
*Interact naturally.* With a global hotkey, COS listens to your intent.
- 💬 **Semantic Search**: Ask "What was I doing with the API yesterday?" and COS finds it.
- 🚨 **Focus Radar**: Detects when you're distracted (YouTube, Twitter) and suggests returning to your high-value tasks.

## 🏗️ Architecture

COS is built on a distributed, low-latency architecture designed for privacy and speed.

| Component | Responsibility | Technology |
| :--- | :--- | :--- |
| **COS AI Core** | OS Monitoring, Screen OCR, Voice Hotkeys | `Python`, `Win32API`, `Tesseract` |
| **COS Backend (Lite)**| Vector Store, Clustering Engine, API | `FastAPI`, `FAISS`, `PyTorch` |
| **COS Desktop UI** | 3D Graph, Timeline, Focus Radar | `React`, `Vite`, `Three.js`, `Tailwind` |
| **COS Extension** | Browser DOM & Context Extraction | `JavaScript`, `Chrome Extension API` |

## 🚀 Getting Started

1. **Clone the Galaxy**:
   ```bash
   git clone https://github.com/Rachit-Kakkad1/HackCruz-COS.git
   ```

2. **Ignite the Brain (Backend)**:
   ```bash
   cd cos-backend-lite
   pip install -r requirements.txt
   python main.py
   ```

3. **Awaken the Senses (AI Core)**:
   ```bash
   cd cos-ai-core
   python core_daemon.py
   ```

4. **Launch the Interface (Desktop UI)**:
   ```bash
   cd cos-desktop-ui
   npm install
   npm run dev
   ```
